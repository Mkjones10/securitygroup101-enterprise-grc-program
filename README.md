# SecurityGroup101 – Enterprise GRC, Risk & Resilience Program

SecurityGroup101 is a rapidly growing fintech company specializing in cloud-based financial platforms, digital payment services, and financial data analytics. As the organization scaled, leadership identified fragmented risk processes, inconsistent controls, and increasing exposure to cyber threats and financial-sector regulations.

This repository simulates a full consulting engagement to design and implement a Governance, Risk, and Compliance (GRC) program aligned to:

- NIST SP 800-30 (Risk Assessment)
- NIST SP 800-34 (Contingency & Disaster Recovery Planning)
- NIST Cybersecurity Framework (CSF)
- ISO 27001
- COBIT 2019
- GLBA, PCI DSS, FFIEC, SOX

The project demonstrates the full lifecycle of a GRC program for a fintech environment: risk quantification, control mapping, compliance analysis, disaster recovery planning, and governance design.
